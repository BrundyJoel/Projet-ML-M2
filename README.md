Documentation technique
Frontend : interface éditeur de texte (React + éditeur riche comme Quill.js)
Backend : API développée avec FastAPI / Flask
Traitement NLP : utilisation de bibliothèques Python (nltk, rapidfuzz)
Architecture : communication entre frontend et backend via API REST


Fonctionnalités IA
Correcteur orthographique : détection et correction des mots mal écrits
Vérification par règles : détection de combinaisons interdites en malagasy
Lemmatisation : extraction de la racine des mots
Autocomplétion : suggestion du mot suivant (N-grams)
Analyse de sentiment : classification simple (positif / négatif)
Traduction mot-à-mot : affichage de la signification d’un mot
Chatbot assistant : aide pour synonymes et suggestions
Synthèse vocale : lecture du texte
